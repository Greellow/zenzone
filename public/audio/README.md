# 音频文件说明

本项目需要以下白噪音音频文件，请将它们放置在 `public/audio/` 目录下：

## 所需音频文件

1. **rain.mp3** - 雨声
   - 推荐来源：https://freesound.org/ 搜索 "rain sound"
   - 或使用：https://www.zapsplat.com/ 免费音效库

2. **forest.mp3** - 森林声（鸟鸣、虫叫）
   - 推荐来源：https://freesound.org/ 搜索 "forest ambience"

3. **fire.mp3** - 篝火声
   - 推荐来源：https://freesound.org/ 搜索 "campfire crackling"

4. **ocean.mp3** - 海浪声
   - 推荐来源：https://freesound.org/ 搜索 "ocean waves"

5. **cafe.mp3** - 咖啡厅环境音
   - 推荐来源：https://freesound.org/ 搜索 "cafe ambience"

6. **whitenoise.mp3** - 白噪音
   - 推荐来源：https://freesound.org/ 搜索 "white noise"

## 快速获取方法

### 方法一：使用 YouTube 下载（需转换为 MP3）
1. 在 YouTube 搜索对应的白噪音视频（如 "rain sounds 1 hour"）
2. 使用在线工具如 https://ytmp3.cc/ 转换为 MP3
3. 重命名并放入 `public/audio/` 目录

### 方法二：使用免费音效网站
访问以下网站下载免费音效：
- https://freesound.org/ （需注册，完全免费）
- https://www.zapsplat.com/ （免费账户）
- https://mixkit.co/free-sound-effects/ （无需注册）

### 方法三：使用现成的白噪音应用导出
如果你有白噪音应用，可以录制音频并导出为 MP3 格式

## 音频要求
- 格式：MP3
- 时长：建议 5-10 分钟以上（会自动循环播放）
- 音质：建议 128kbps 或更高
- 大小：单个文件建议不超过 10MB

## 临时测试
如果暂时没有音频文件，应用仍可正常运行，只是点击场景时不会播放声音。
可以先使用任意 MP3 文件进行测试。
