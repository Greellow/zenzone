# 静谧空间 (ZenZone) - 需求分析文档

## 一、项目概况

### 1.1 项目基本信息
- **项目名称**：静谧空间 (ZenZone)
- **项目类型**：移动端专注/助眠 Web 应用
- **开发技术**：Vue 3 + Vant UI + TypeScript
- **目标平台**：移动端浏览器（iOS Safari、Android Chrome、微信浏览器）
- **开发时间**：2024年12月

### 1.2 项目背景
在当今快节奏的学习和工作环境中，学生和职场人士面临着以下挑战：
- 学习压力大，难以集中注意力
- 环境噪音干扰，影响专注效率
- 睡眠质量差，入睡困难
- 缺乏有效的时间管理工具

本项目旨在开发一款基于移动端的网页应用，通过白噪音播放和番茄钟倒计时功能，帮助用户：
- 屏蔽环境干扰，提升专注力
- 改善睡眠质量，快速进入放松状态
- 培养良好的时间管理习惯

## 二、用户需求分析

### 2.1 目标用户群体
1. **学生群体**：需要长时间专注学习，准备考试
2. **职场人士**：需要高效工作，管理时间
3. **失眠人群**：需要放松助眠，改善睡眠质量
4. **冥想爱好者**：需要安静的环境进行冥想练习

### 2.2 用户痛点
- 环境嘈杂，无法专注
- 时间管理困难，拖延症严重
- 入睡困难，睡眠质量差
- 现有应用功能复杂，学习成本高

### 2.3 用户期望
- 界面简洁美观，操作简单
- 白噪音种类丰富，音质优良
- 倒计时功能准确可靠
- 支持深色模式，夜间使用不刺眼
- 无需下载安装，打开即用

## 三、功能需求

### 3.1 首页 - 声景选择

#### 3.1.1 功能描述
用户可以浏览和选择不同的声音场景，点击即可播放对应的白噪音。

#### 3.1.2 声景列表
1. **雨夜** 🌧️ - 淅淅沥沥的雨声，营造宁静氛围
2. **森林** 🌲 - 鸟鸣虫叫，感受大自然的气息
3. **篝火** 🔥 - 噼啪作响的火焰，温暖舒适
4. **海浪** 🌊 - 海浪拍打沙滩，聆听大海的呼吸
5. **咖啡厅** ☕ - 咖啡厅环境音，营造专注氛围
6. **白噪音** 🎵 - 纯粹的白噪音，屏蔽一切干扰

#### 3.1.3 交互设计
- 宫格布局展示所有声景
- 点击声景图标播放/暂停音频
- 背景图片随选中的声景动态切换
- 显示当前播放的声景信息
- 支持音频循环播放

### 3.2 专注页 - 番茄钟倒计时

#### 3.2.1 功能描述
提供番茄钟倒计时功能，帮助用户进行时间管理和专注训练。

#### 3.2.2 核心功能
- **时长选择**：支持 15/25/45/60 分钟预设时长
- **倒计时器**：大尺寸圆形倒计时显示
- **进度显示**：实时显示倒计时进度百分比
- **控制功能**：开始、暂停、继续、重置
- **音频联动**：倒计时期间持续播放选中的白噪音
- **完成提醒**：倒计时结束时播放提示音和通知

#### 3.2.3 番茄工作法
- 专注工作 25 分钟
- 短暂休息 5 分钟
- 重复 4 次后，长休息 15-30 分钟

### 3.3 设置页 - 个人中心

#### 3.3.1 音量控制
- 滑块调节全局音量（0-100%）
- 实时预览音量变化
- 音量设置持久化保存

#### 3.3.2 主题切换
- 日间模式：明亮清爽的配色
- 深色模式：护眼舒适的暗色调
- 自动检测系统主题偏好
- 主题设置持久化保存

#### 3.3.3 使用说明
- 如何使用应用
- 白噪音科普知识
- 番茄工作法介绍

#### 3.3.4 关于我们
- 应用名称和版本信息
- 开发团队成员介绍
- 版权信息

## 四、非功能需求

### 4.1 性能要求
- 页面加载时间 < 2 秒
- 音频播放流畅，无卡顿
- 倒计时精确到秒
- 场景切换响应时间 < 500ms

### 4.2 兼容性要求
- iOS Safari 12+
- Android Chrome 80+
- 微信内置浏览器
- 支持主流移动设备分辨率

### 4.3 可用性要求
- 界面简洁直观，无需学习即可使用
- 操作流畅，符合移动端交互习惯
- 提供清晰的视觉反馈
- 支持触摸手势操作

### 4.4 可靠性要求
- 音频播放稳定可靠
- 倒计时准确无误
- 数据持久化保存（音量、主题设置）
- 异常情况下不影响核心功能

## 八、风险分析

### 8.1 技术风险
- **音频兼容性**：不同浏览器对音频格式支持不同
  - 解决方案：使用 MP3 格式，兼容性最好
- **通知权限**：用户可能拒绝通知权限
  - 解决方案：提供备选提示方式

### 8.2 用户体验风险
- **音频资源加载慢**：网络不佳时音频加载慢
  - 解决方案：提供加载提示，优化音频文件大小
- **电池消耗**：长时间播放音频消耗电量
  - 解决方案：提醒用户注意电量
